# Mega Man Maker batch level downloader
A tool to download multiple Mega Man Maker level times at once!

## How to run
1. Clone the project
```
git clone https://github.com/midmag/mmm-level-batch-downloader.git
```

2. Install the required packages
```
pip install -r requirements.txt
```

3. Run the script
```
python batchDownloader.py
```
